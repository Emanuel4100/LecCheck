# 🎓 LecCheck - Lecture Tracker

[![Web Version](https://img.shields.io/badge/Web_Version-Live-success?style=for-the-badge&logo=google-chrome)](https://emanuel4100.github.io/LecCheck/)
[![Python](https://img.shields.io/badge/Python-3.11-blue?style=for-the-badge&logo=python)](https://www.python.org/)
[![Flet](https://img.shields.io/badge/Built_with-Flet-0080FF?style=for-the-badge)](https://flet.dev/)

LecCheck is a cross-platform application (Web & Android) designed to help students efficiently track, manage, and monitor their university lectures, practices, and labs. 

## ✨ Features
* **☁️ Cloud Sync:** Seamlessly syncs your schedule across all devices using **Firebase Realtime Database**.
* **🔐 Google Authentication:** Secure login using your Google Account (OAuth2) via **Google Cloud Run**.
* **🌍 Multi-Language:** Built-in support for English and Hebrew (RTL).
* **📱 Cross-Platform:** Works directly in the browser via WebAssembly (GitHub Pages) or as a native Android App (APK).
* **📊 Statistics:** Visual insights into your study progress and upcoming/pending lectures.

## 🔗 Try it Live!
Access the web version directly from your browser (No installation required):
👉 **[LecCheck on GitHub Pages](https://emanuel4100.github.io/LecCheck/)**

## 🛠️ Tech Stack
* **Frontend UI:** [Flet](https://flet.dev/) (Flutter for Python)
* **Backend Server:** Google Cloud Run (Dockerized)
* **Web Hosting:** GitHub Pages
* **Database:** Firebase
